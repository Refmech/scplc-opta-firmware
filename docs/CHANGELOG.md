## Releases

When cutting a release:
- Update the semantic version values mirrored into Modbus holding registers (HR7..HR9).
- Create an annotated Git tag (example: `opta-vYYYY.MM.DD` or `vMAJOR.MINOR.PATCH`).
- Optionally create a GitHub Release from that tag.
